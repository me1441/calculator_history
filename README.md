# 🧮 Калькулятор с историей операций

Графический калькулятор на Python с использованием `tkinter`, поддерживающий историю вычислений, сохранение в JSON и экспорт в текстовые файлы.

---

## 📋 Описание

Приложение представляет собой стандартный калькулятор с расширенной функциональностью:
- Выполнение базовых арифметических операций (+, −, ×, ÷)
- Автоматическое сохранение истории всех операций
- Просмотр истории в удобном формате с временными метками
- Экспорт истории в текстовый файл
- Персистентное хранение данных в JSON

---

## 🚀 Запуск

```bash
python calculator_history.py
```

**Требования:** Python 3.7+

**Зависимости:** только стандартная библиотека Python (`tkinter`, `json`, `datetime`)

---

## 🎮 Инструкция по использованию

### Основные операции
1. Введите первое число с помощью цифровой клавиатуры или клавиш
2. Выберите операцию в выпадающем списке (+, −, *, /)
3. Нажмите **"Вычислить"** или клавишу `Enter`
4. В появившемся диалоге введите второе число и подтвердите
5. Результат отобразится на дисплее, а операция — в истории

### Управление дисплеем
| Кнопка / Клавиша | Действие |
|------------------|----------|
| `0–9`            | Ввод цифр |
| `.`              | Десятичная точка |
| `±`              | Смена знака |
| `BackSpace`      | Стереть последний символ |
| `Escape`         | Очистить дисплей |
| `Enter`          | Вычислить |

### История операций
- Все вычисления автоматически сохраняются с временной меткой
- Нажмите **"🗑️ Очистить историю"** для удаления всех записей
- Нажмите **"💾 Сохранить в файл"** для экспорта в `.txt`

## 📁 Структура файлов

```

```
